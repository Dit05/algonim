# About

Welcome to Algonim's documentation!

When writing animation functions in the web editor, you have access to {@link Algonim} through the `Algonim` object, and an instance of {@link Sequence} is passed as the first argument.

You can also look at the built-in examples, selectable with the drop-down on the editor page.

# What you can do

Algonim is a decently capable declarative animation framework with its own GIF encoder.

- Built-in models have some customization options.
- You can make your own models by extending {@link models/Model.Model | Model}.
- For small tasks like drawing some text or arrows, you can also use an instance of the {@link models/Overlay.Overlay | Overlay} model.
